SOLUSI WAJIB HARI 3
1. Timeout Per Router

Misal:

max 5 detik

kalau lebih → skip

2. Circuit Breaker

Jika router gagal 5x berturut:

Stop polling 5 menit

Jangan retry tiap 10 detik

3. Backpressure Control

Jika DB write time > threshold:

Kurangi concurrency

Jangan terus push log
