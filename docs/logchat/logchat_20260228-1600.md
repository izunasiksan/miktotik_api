# Update: 2026-02-28 - Export Reports & Date Filter
**Kategori:** Reporting Feature

## Ringkasan Perubahan
Fitur Reporting ditingkatkan dengan kemampuan untuk mengekspor data ke format PDF dan CSV, serta melakukan filter data berdasarkan rentang tanggal kustom (Date Range Picker).

## Detail Implementasi
### 1. Backend (app/api/endpoints/reports.py)
* **Endpoint Baru:** `/export/{board_id}` untuk generate PDF/CSV.
* **Modifikasi Endpoint:** `/daily/{board_id}` dan `/monthly/{board_id}` menerima parameter opsional `start_date` dan `end_date`.
* **Dependency:** Menggunakan `reportlab` untuk PDF generation.

### 2. Frontend (src/pages/Reports.jsx)
* **UI Update:** Menambahkan input `start_date` dan `end_date`.
* **Functionality:** Tombol Export CSV dan PDF memanggil endpoint `/export`.
* **State Management:** Fetch data ulang saat tanggal berubah.

## Pengujian
* **PDF Export:** Berhasil menghasilkan file `.pdf` dengan tabel laporan.
* **CSV Export:** Berhasil menghasilkan file `.csv` dengan header yang sesuai.
* **Date Filter:** Data laporan difilter dengan benar sesuai rentang tanggal.

## Catatan Penting
* Library `reportlab` ditambahkan ke `requirements.txt`.
* Endpoint export menggunakan streaming response untuk efisiensi memori.
